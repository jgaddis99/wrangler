When to Choose Wrangler

  • You use ultrawide monitors and need configurable grids, not just halves/quarters
  • You run many terminal windows and want to batch-tile them with one action
  • You switch between Mac and Windows keyboards and want consistent shortcuts
  • You want to move windows directionally through a grid, cell by cell
  • You want a visual overlay that shows your actual desktop layout
  • You want to pin apps to fixed positions and reset with one key

When to Choose Something Else

Rectangle if you want a simple, proven tool with drag-to-edge snapping and don't need grid customization.

Amethyst if you want automatic tiling (windows rearrange themselves) rather than manual grid placement.

Moom if you want a polished commercial app with a track record and App Store support.

Frequently Asked Questions

Is Wrangler really free?

Yes. Free, open source, MIT licensed. No trial, no premium tier, no account required.

Does Wrangler work with ultrawide monitors?

Yes. Wrangler was built specifically for ultrawide setups. You can configure any grid size per display (e.g., 6 columns for a 49" ultrawide) and snap to thirds, which is essential for ultrawide workflows.

Does Wrangler work with multiple monitors?

Yes. Each display has its own independent grid configuration. Arrow keys move windows across monitor boundaries automatically when you reach the edge of a grid.

Can I tile all my terminal windows at once?

Yes. Open the grid overlay (Ctrl+Alt+Space), hold Cmd, and drag across a zone. All windows of the focused app tile evenly into that area. Works with any app, not just terminals.
