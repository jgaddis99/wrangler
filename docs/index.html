Wrangle your windows.

A macOS window manager that snaps windows to configurable grids. Built for developers and power users who run multiple terminals across ultrawide monitors.

Download for macOS View on GitHub
Features

Everything you need to manage windows.

⬜

Grid Snapping

Move windows one cell at a time with arrow keys. Grow or shrink by adding cells. Snap to halves, quarters, or thirds instantly.

🖼

Visual Overlay

See all your displays with wallpapers and grid lines. Drag to select any zone. Live preview shows exactly where the window will land.

⚡

Batch Tile

Cmd+drag on the overlay to tile all windows of an app at once. Perfect for organizing a dozen terminal windows in seconds.

📌

App Pinning

Pin Slack, Mail, or Discord to fixed positions. One shortcut resets all pinned apps to their designated spots when things get messy.

⌨

Cross-Keyboard

Uses Ctrl+Alt as the base modifier — same physical keys on Mac and Windows keyboards. Switch between keyboards without relearning shortcuts.

🔄

Auto-Update

Built-in update checking via Sparkle. Get new features automatically without checking GitHub. No account or subscription required.

Keyboard Shortcuts

Learn once, use forever.

ShortcutAction
Ctrl+Alt+ArrowMove window one grid cell
Ctrl+Alt+Shift+ArrowGrow window one cell in that direction
Ctrl+Alt+U I J KSnap to quarter (spatial 2x2 on keyboard)
Ctrl+Alt+1 2 3Snap to left / center / right third
Ctrl+Alt+EnterMaximize
Ctrl+Alt+CCenter on display
Ctrl+Alt+TAuto-tile all windows on display
Ctrl+Alt+ZUndo last snap
Ctrl+Alt+SpaceOpen visual grid overlay
Ctrl+Alt+RReset all pinned apps
Ctrl+Cmd+ArrowMove to adjacent display

All shortcuts are fully customizable in Settings.
